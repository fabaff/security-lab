Fedora Security Lab

Das Fedora Security Lab stellt eine sichere Testumgebung für Sicher- heitsprüfung und für den Unterricht von Sicherheitstestmethodiken an Universitäten oder anderen Organisationen zur Verfügung.


Das Fedora Security Lab ist ein offizieller Fedora Spin.

  • Live CD
  • Runs from USB devices
  • LXDE_desktopumgebung
  • über 70 Werkzeuge
  • Angepasste Menü-Struktur
  • installierbar
  • verfügbar seit Fedora 13
  • 6-monatiger Veröffentlichungszyklus
  • zuerst OpenBox, dann LXDE
  • konstante Weiterentwicklung
